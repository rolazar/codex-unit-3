# Component Lifecycle

## Objective

Learn about the lifecycle of React components.

## Benefits

Understanding the component lifecycle will help you understand the different phases when a component can be changed or affected.

## Complete these tasks

1. In the terminal, navigate to the `react-lifecycle` folder in this level folder.
2. Start the Vite server and view the page in the browser. You may need to install `node_modules`.
3. Read the information in the webpage.
4. Try the buttons.
5. Inspect the DOM.
6. View `App.jsx` and `Main.jsx` (in `src/components/`). Observe how they use React variables, state, and props.
7. In `script.js`, create an array of messages.
8. Let the messages explain the lifecycle of React components. Use 1 string per sentence. Use your own words. Don't copy/paste.
9. Use `console.log` to display the array.
10. Use `node` to run `script.js`.

## More Information

- The lifecycle of React components consists of 3 phases - mount, update, and unmount.
- A component mounts when it's added to the DOM.
- A component updates when it changes in the DOM, usually by receiving new props or state.
- A component unmounts when it's removed from the DOM.
